Hallo, 

In dieser Studie können Sie 16 Spiele spielen. Durch Drücken der "Page Down" Taste gelangen Sie jeweils zum nächsten Spiel.

Zuerst gibt es eine kurze Einführung.

In jedem Spiel werden mehrere Spielautomaten gezeigt. Wenn Sie an einem Spielautomaten spielen, können Sie "1 Punkt" gewinnen. 

Ihre Aufgabe ist es, so viele Punkte wie möglich zu gewinnen. Dafür können Sie zwischen verschiedenen Spielautomaten wählen. Sobald Sie einen Spielautomaten auswählen, verdienen Sie entweder 1 Punkt oder 0 Punkten.
